# 广州东振官网 (dzweb) 产品需求文档 (PRD)

## 1. 产品定位
本系统为 **广州东振机电设备有限公司** 的官方门户网站
，集成“企业品牌展示”、“产品目录管理”、“人才招聘系统”及“内部办公辅助（加班管理/应用导航）”于一体。

## 2. 核心用户场景
- **外部客户**：浏览公司自动化方案、下载产品资料、在线留言。
- **求职者**：查看最新招聘岗位、提交应聘意向。
- **内部员工**：快速访问内部系统导航、提交/查询加班记录、管理产品库。
- **管理员**：审核留言、发布产品、更新招聘信息、管理用户权限。

## 3. 功能模块
### 3.1 门户展示 (Public)
- **首页 (Home)**：公司简介、发展历程、组织架构、战略与风格。
- **产品中心 (Product)**：多级分类展示（机器人、自动化设备、夹具等），支持详情查看。
- **成功案例 (Case)**：展示工业机器人焊接、组装线等实际工程案例。
- **关于东振 (About)**：企业文化与实力展示。

### 3.2 互动与招聘 (Engagement)
- **联系我们 (Contact)**：留言板（Message Board）、地理位置引导、邮箱。
- **加入我们 (Human)**：招聘岗位发布、更新与管理。

### 3.3 内部管理系统 (Internal - Auth Required)
- **应用中心 (App Central)**：整合公司内部各业务系统链接，作为统一入口。
- **加班管理 (Overtime)**：针对特定业务（周末加班等）的记录与权限管理。
- **产品管理**：管理员可进行产品的创建、分类更新及图片上传。**物理资源同步**：产品记录的删除或更新将同步清理服务器上的旧/物理图片，防止存储空间冗余。

## 4. 非功能性需求
- **多语言支持**：支持 中、英、日 三语切换（Flask-Babel）。
- **静态资源管理**：大量案例图片与产品说明书的分类存储。
- **存储效率与自动维护**：提供 CLI 工具（`flask cleanup-images`）自动识别并删除数据库中无记录的“孤儿”文件，确保生产服务器的存储健康。
- **安全性**：基于 Session 的用户权限校验，敏感操作（如删除产品、管理用户）需特定权限。
