# 广州东振官网 (dzweb) 产品需求文档 (PRD)

## 1. 产品定位
本系统为 **广州东振机电设备有限公司** 的官方门户网站，集成“企业品牌展示”、“产品目录管理”、“人才招聘系统”及“内部办公应用集成”于一体。

## 2. 核心用户场景
- **外部客户**：浏览公司自动化方案（机器人焊接、非标自动化设备、工装夹具、视觉系统集成）、下载产品资料、在线留言。
- **求职者**：查看最新招聘岗位、提交应聘意向。
- **内部员工**：快速访问内部系统导航、管理产品库、处理客户咨询。
- **管理员**：审核留言、发布/更新产品、维护招聘信息。

## 3. 功能模块
### 3.1 门户展示 (Public)
- **首页 (Home)**：公司简介、机器人焊接方案展示、发展历程。
- **产品中心 (Product)**：支持 WebP 优化、网格化展示、多级分类（机器人、自动化设备、夹具等）。
- **成功案例 (Case)**：展示工业机器人集成实际工程案例。

### 3.2 互动与招聘 (Engagement)
- **联系我们 (Contact)**：留言板（Message Board）、反馈收集（已移除物理地理位置展示以简化页面并保护隐私）。
- **加入我们 (Human)**：招聘岗位动态发布。

### 3.3 内部管理系统 (Internal - Auth Required)
- **应用中心 (App Central)**：整合公司内部各业务系统链接，作为统一入口（已移除过时的应用密码管理）。
- **产品管理**：管理员可进行产品的创建、分类更新及图片上传。**物理资源同步**：产品记录的删除或更新将同步清理服务器上的旧图片及对应缩略图。

## 4. 非功能性需求
- **SEO & 性能**：
  - **图像优化**：上传图片自动生成 **缩略图 (Thumbnail)** 并在列表页优先使用，显著减少首屏加载时间。
  - **SEO 增强**：支持自动生成的 `sitemap.xml` 和 `robots.txt`，集成百度主动推送 (Baidu Active Push) 以加快页面收录。
- **多语言支持**：支持 中、英、日 三语切换（Flask-Babel）。
- **存储效率**：提供 CLI 工具（`flask cleanup-images`）自动清理孤儿文件（原始图与缩略图）。
- **安全性**：基于环境变量密码的管理员 Session 校验，敏感操作（如删除产品）需管理员权限。删除操作采用静默处理。
