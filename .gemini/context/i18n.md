# Internationalization (I18n) Strategy (English)

## Tech: Flask-Babel
The system supports Chinese (zh), English (en), and Japanese (ja).

## Implementation Details
1. **Extraction**: Marker `_l()` or `gettext` in Jinja2 templates and Python code.
2. **Translation Files**:
   - `dzweb/translations/en/LC_MESSAGES/messages.po`
   - `dzweb/translations/ja/LC_MESSAGES/messages.po`
3. **Locale Selection**:
   - Defined in `dzweb/lang.py`.
   - Priority: User-selected cookie or browser accept-language.

## Workflow for AI
When adding new features:
1. Wrap user-facing strings in `_l()` or `gettext`.
2. Ensure translations are updated using `pybabel` CLI commands.
