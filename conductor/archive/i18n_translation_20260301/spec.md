# Track Specification - i18n_translation_20260301

## Overview
广州东振官网（dzweb）需要全面完善英语（en）与日语（ja）翻译。本次任务旨在填补所有已提取但尚未翻译的空条目（msgstr ""），确保核心 UI、首页简介、产品展示及案例展示的国际化内容完整且专业。

## Functional Requirements
- **识别空条目**：使用 `pybabel` 识别所有在 `en` 和 `ja` 分支下尚未翻译的字符串。
- **完善核心 UI 翻译**：翻译导航栏、页脚、各页面标题及通用按钮文案。
- **完善内容翻译**：完成“首页/关于我们”、“产品展示”及“经典案例”部分的全文翻译。
- **专业术语一致性**：建立并应用中-英-日专业术语库（如“机器人焊接”、“挤出机”等），确保全站术语统一。
- **本地化适配**：针对英语和日语受众调整部分表达，使其符合当地文化及工业术语习惯。

## Non-Functional Requirements
- **准确性**：所有翻译需符合工业自动化行业的专业严谨性。
- **排版兼容**：确保英语和日语的长文案在 Jinja2 模板中显示完整，不破坏布局。
- **持续维护**：更新后的 `.po` 文件需重新编译为 `.mo` 文件以生效。

## Acceptance Criteria
- [ ] `pybabel` 扫描结果显示 `en` 和 `ja` 的 `.po` 文件中不再包含空的 `msgstr`。
- [ ] 切换至英语版后，全站核心内容、导航及产品描述均显示为正确英语。
- [ ] 切换至日语版后，全站核心内容、导航及产品描述均显示为正确日语。
- [ ] 专业术语（如机器人、挤出机等）在不同页面间翻译一致。

## Out of Scope
- 添加新的功能模块或修改前端 CSS 样式。
- 对第三方外链内容进行翻译。
