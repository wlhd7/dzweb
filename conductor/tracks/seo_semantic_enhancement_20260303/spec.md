# SEO 结构优化与语义化增强 规范文档

## 1. 概述 (Overview)
本项目旨在优化广州东振机电设备有限公司网站的 HTML 语义化结构。主要通过添加 <h1> 标签、完善图片 alt 属性和优化列表页标题，在不改变视觉设计的前提下，提升搜索引擎（百度、Google 等）的权重和收录效果。

## 2. 功能要求 (Functional Requirements)
- **H1 语义化 (核心页面)**: 
  - **首页**: 增加包含公司名称和核心业务的 <h1> 标签。使用 CSS 视觉隐藏技术确保用户不可见，但对爬虫可见。
  - **产品详情页**: 将当前的产品名称标题 (<h2>) 升级为 <h1>。
- **图片 Alt 属性 (基础)**: 
  - 首页、侧边栏、分类列表页的所有装饰性或功能性图片均需增加 `alt` 属性。内容应自动提取对应产品的名称、分类或描述性的翻译文本。
- **列表页标题优化**: 
  - 产品分类页面（如非标零件、工装夹具等）的标题由 <p> 标签升级为 <h1>。

## 3. 非功能要求 (Non-Functional Requirements)
- **视觉一致性**: 严禁修改现有的视觉布局、颜色、字体或间距。
- **代码重用**: 优先修改 `base.html`、`product/main.html` 和 `product/display.html` 等核心模板，避免重复开发。
- **多语言兼容**: 所有的 <h1> 内容和 `alt` 文本必须支持 {{ _('...') }} 国际化。

## 4. 验收标准 (Acceptance Criteria)
- [ ] 首页源码中包含且仅包含一个 <h1> 标签，视觉上保持不变。
- [ ] 所有产品详情页的主标题均使用 <h1>。
- [ ] 产品分类列表页的主标题均使用 <h1>。
- [ ] 全站主要图片均已具备有意义的 `alt` 描述文本。
- [ ] 通过 W3C 验证（可选）并确保 <h1> 结构符合 SEO 最佳实践。
