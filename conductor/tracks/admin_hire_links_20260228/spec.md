# Track Specification - admin_hire_links_20260228

## Overview
此任务旨在为“招贤纳士”页面（`human.hire`）的招聘列表项添加管理员管理链接。当管理员已通过系统登录时，每个职位描述的底部应显示“编辑”和“删除”的蓝色文本链接，以便快速管理招聘信息。

## Functional Requirements
- **权限验证**：页面应基于 Session 中的管理员登录状态进行判断。
- **动态显示**：仅在管理员已登录的情况下，在招聘列表的每个职位项底部显示管理链接。
- **编辑功能**：点击“编辑”应跳转至现有的编辑页面 (`human.update_position`)。
- **删除功能**：点击“删除”时应触发浏览器自带的确认对话框 (`confirm('...')`)。确认后执行删除操作。
- **国际化 (I18n)**：所有新增链接文本（“编辑”、“删除”）及确认提示语必须使用 `{{ _('...') }}` 进行国际化处理。

## Acceptance Criteria
- [ ] 普通用户访问时看不到“编辑”和“删除”链接.
- [ ] 管理员登录后访问，每个职位的“职位要求”下方显示蓝色文本形式的“编辑”和“删除”链接。
- [ ] 点击“编辑”链接能正确进入该职位的编辑表单。
- [ ] 点击“删除”链接会弹出确认提示。确认后，职位被删除并刷新列表。
- [ ] 链接和提示文案在不同语言切换下显示正确。

## Out of Scope
- 修改招聘信息的数据库结构或后端 CRUD 逻辑（假设已存在）。
- 重新设计管理员登录系统。
